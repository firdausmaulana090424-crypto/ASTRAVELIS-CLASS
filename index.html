Cara Perbarui Gambar Jadwal

Untuk mengganti file jadwal pelajaran tanpa perlu input ketik satu-satu lagi:

  1. Simpan atau eksport file jadwal baru kamu menjadi format gambar png/jpg.
  2. Beri nama file tersebut dengan nama jadwal-x3.png
  3. Upload file gambar baru tersebut ke repositori GitHub kamu untuk menggantikan file lama.
